# RUN_PROGRESSION.md — Rune Fencer Tank Progression (SECONDARY GOAL)

> **Goal:** level RUN from 81 → 99 and gear it as a **tank** (magic-tank: damage taken down, magic
> evasion, enmity) to a "can contribute to endgame content" state. **Tank role only** — this guide
> ignores DD/DPS sets entirely.
>
> **Companion docs:** daily activities in [`ACCOUNT_PROGRESSION.md`](./ACCOUNT_PROGRESSION.md); your
> main job in [`BLU_PROGRESSION.md`](./BLU_PROGRESSION.md). RUN's reforged tank pieces ride the **same
> content gates as BLU** — that overlap is called out throughout.
>
> **Sourcing note:** verified against current (mid-2026) sources where reachable (BG-wiki Community
> Rune Fencer Guide — last edited June 2026, FFXIAH RUN guides). FFXI wikis are bot-protected, so some
> figures came via search excerpts. Volatile items flagged **[CONFIRM in-game]**; departures from the
> original brief flagged **▶ Brief correction**.

## Section A — Leveling 81 → 99

- **No cap quests needed.** All limit breaks are account-wide and you have BLU 99, so RUN levels on
  **straight XP** to 99.
- **Method: trust-carry.** Sub **/WAR** — Provoke to practice enmity, Berserk/Double Attack to speed
  kills while leveling.
- **Trust party for XP:** **Valaineral** (let it tank so you practice your kit safely, or self-tank to
  rehearse enmity) + **Cherukiki** (heals) + **Koru-Moru** (haste/refresh) + **Qultada** (COR rolls) +
  **Kupofried** (XP-bonus aura) + a DD/SC trust (Iroha II / Tenzen II / Lion II).
- **Stack XP multipliers:** **Echad Ring** (Dedication) + **Kupofried** + **Corsair roll** + **RoV
  XP bonus** + **Signet**. Run **Records of Eminence** XP objectives alongside.
- **Camps by band:** continue in level-appropriate zones now; at **80+** move to **Escha – Zi'Tah**
  (Opo-opos / Yztargs) for fast, dense XP. **[CONFIRM current best camp in-game]** — campaign bonuses
  shift the "best" spot.

### Tank tools by level (what comes online as you climb)
| Level | Ability | Role |
|-------|---------|------|
| 10 | Vallation | magic damage reduction |
| 20 | Swordplay | accuracy/evasion ramp |
| 40 | Pflug | resist/status defense |
| 50 | Valiance | party-wide magic mitigation |
| 75 (merit) | Battuta / Rayke | parry-counter / resist-down on enemy |
| **85** | **Liement** | **absorb magic damage — the "now it feels like a tank" threshold** |
| 95 | One for All | party damage-absorb shield |
| 96 | Odyllic Subterfuge | enemy magic-immunity-strip / big enmity utility |

**Takeaway:** RUN functions as a real magic-tank from **Liement at 85**, and is fully online by
**96**. Push to **99** for the great-sword weaponskill (Dimidiation) and the full tool kit.

## Section B — Unlock quests

- **Children of the Rune** (the RUN job unlock) — **almost certainly already done** if you're 81;
  **[CONFIRM]**. (Western Adoulin; Yahse Hunting Grounds wildflower petal → Octavien rune-enhancement
  spam.) **Keep the Sowilo Claymore** it gives you — it's used in a later AF-related quest.
- **Dimidiation** (the great-sword weaponskill) — **NOT done; it's a level-99 quest.** Quest "Rune
  Fencing the Night Away," **Gaddiux (Western Adoulin, J-10)**. You get a **Trial Blade**, build WS
  points with it (use a Light skillchain — e.g. **Trust: Ayame** for Resolution → Light, farm in
  **Kamihr Drifts**), then trade it back. **Do this at 99** — Dimidiation is RUN's tanking-while-doing-
  damage WS and you want it before joining content.

## Section C — Tank gear (no RUN armor yet)

**Architecture (read this first):** RUN tank sets are built from **reforged Empyrean/Artifact JSE
(Erilaz / Futhark)** + **Nyame** + tank accessories — **not** from an Ambuscade armor set.

▶ **Brief correction (Ayanmo):** **Ayanmo is a BLU DD set — wrong stats for tanking.** Do not build
Ayanmo for RUN. The Ambuscade *armor* doesn't anchor a RUN tank; the only Ambuscade piece that matters
for RUN is the **cape** (below).

Priority order, cheapest floor → endgame:

### 1. Day-one floor at 99 — Sparks / RoE DT set
- **What:** a full **Sparks/Records-of-Eminence iL119 set** tuned toward **Damage Taken (DT)**.
- **Source:** Sparks of Eminence vendors — instantly buyable, no gates.
- **Role:** "allowed in the building" gear. Gets you tanking *immediately* at 99 while the real sets
  come together.

### 2. Ogma's Cape (augmented) — your best easy piece
- **What:** the **Ambuscade cape**, augmented **DT + enmity (+HP)** for tanking.
- **Source:** redeem an **Ambuscade Voucher: Back** from **Gorpa-Masorpa (Mhaura, G-9)**, then augment
  it by trading it back with **Abdhaljs materials** (Thread/Dust/Sap/Dye/Resin — each unlocks a
  different augment line). Build the **HP / DT / Enmity** lines, not DEX/Attack.
- **Why:** reachable purely from the **Ambuscade you already run daily** — no new gates.
  → [`ACCOUNT_PROGRESSION.md` → Ambuscade](./ACCOUNT_PROGRESSION.md#1-ambuscade--the-non-negotiable-daily).
- **[CONFIRM the current best tank augment combination in-game]** — Abdhaljs augment pools rotate.

### 3. A functional great sword
- **Floor:** an **Ambuscade great sword** to start.
- **Standard tank GS — Aettir (Oboro):** **+10 Enmity, Physical Damage Taken II −5%**, and with the
  Oboro augment a large **accuracy + magic-evasion** boost. This is the current community-standard RUN
  tank great sword. **The grind is real** (Magian trial chain + ~300 Pluton / Beitetsu / Riftborn
  Boulder, then Oboro augment) — it's a project, not a quick buy.
- **Capstone — Epeolatry (Ergon / Aeonic):** the endgame RUN great sword; it **upgrades Liement to an
  AoE (10-yalm) enmity effect**, scaling enmity per party member hit. Long-term, group-assisted (Aeonic
  weapons are a major undertaking).
- **▶ Brief note (Aettir vs Daihate):** the brief suggested **Daihate** (an Ambuscade GS, no Pulse
  weapon needed) as a pragmatic casual pick. Current sources center the tank conversation on **Aettir**
  (and Macbain for DD, Humility for hybrid), with Epeolatry as capstone. **Recommendation:** use the
  **Ambuscade GS as your floor**, then commit to **Aettir** as the real tank weapon when you want the
  project; treat **Daihate as an acceptable budget stopgap** if you don't. **[CONFIRM the current best
  tank GS in-game]** — weapon meta shifts with updates.

### 4. Reforged RUN JSE — the real tank sets
These ride the **same reforge ladder and gates as your BLU pieces** — see
[`BLU_PROGRESSION.md` → reforge ladder](./BLU_PROGRESSION.md#the-reforge-ladder-how-every-piece-moves).
You hold **12 of every Rem's Tale chapter**, so chapters are covered for RUN too.

- **Futhark (reforged Relic)** — **Bandeau (head), Torque (neck-ish JSE), Mitons (hands)** form the
  enmity core. **Futhark Mitons +3/+4** are excellent enmity hands. **Source:** the **same
  Dynamis-Divergence path** as BLU Relic; **no Vagary title needed** for Relic.
  → [`ACCOUNT_PROGRESSION.md` → Dynamis-D](./ACCOUNT_PROGRESSION.md#4-dynamis-divergence--relic-23-fuel-blu).
- **Erilaz (reforged Empyrean)** — the JSE tank **legs + feet**.
  ▶ **Brief correction:** the brief said Erilaz comes from "Omen (AF reforge)." It does **not** —
  **Erilaz is reforged *Empyrean*, and its +2 comes from Sortie** (10,000 Gallimaufry + 1 Ra'Kaznar
  Sapphire per piece), exactly like your BLU Empyrean pieces. Its **109→119 step needs the Vagary
  title** (same SoA gate as BLU).
  → [`ACCOUNT_PROGRESSION.md` → Sortie](./ACCOUNT_PROGRESSION.md#3-sortie--empyrean-23-fuel--the-empyrean-earring-blu).
- **Tank accessories (build alongside):**
  - **Neck:** Unmoving Collar +1 (Unity).
  - **Ears:** Odnowa Earring +1 and/or Tuisto Earring (HP/DT).
  - **Rings:** an enmity ring (Begrudging / Pernicious / Provocare) + a DT ring.
  - **Waist:** Platinum Moogle Belt or Goading Belt (enmity).
  - **Body (interim):** Emet Harness +1 / Adamantite Armor before Nyame/Erilaz are ready.
  - **[CONFIRM current accessory BiS in-game]** — these rotate; the above are the durable staples.

### 5. Endgame tank — Nyame, then Epeolatry
- **Nyame armor set** (augmented) — a **massive survivability leap** and the base set used to tank most
  of the game. **Source:** **Odyssey** (Sheol C Lustreless mats → augments; the best comes from
  **Gaol/Bumba**, a group farm). A common tank baseline is **Nyame Rank 25, Path B**. **[CONFIRM the
  current recommended Nyame path in-game]**.
  → [`ACCOUNT_PROGRESSION.md` → Odyssey](./ACCOUNT_PROGRESSION.md#5-odyssey--long-term-nyame--a-strong-gil-source-both-jobs).
- **Epeolatry** (see GS, above) as the capstone weapon. Long-term.

## Section D — Gates & dependencies (and how they overlap BLU)

| RUN piece | Gate | Shared with BLU? |
|-----------|------|------------------|
| Sparks DT floor | none | — (immediate) |
| Ogma's Cape | Ambuscade (daily) | shares the Ambuscade farm |
| Ambuscade GS | Ambuscade | shares the Ambuscade farm |
| **Erilaz (Empyrean) 109→119** | **Vagary title** | **YES — same SoA ~5-3-2 unlock as BLU Empyrean** |
| Erilaz +2 | Sortie Gallimaufry + Sapphire | shares your BLU Sortie currency pool |
| **Futhark (Relic) +2** | **Dynamis-Divergence** | **YES — same Dyna-D unlock as BLU Relic** |
| Nyame | Odyssey | shares your Odyssey runs |
| Aettir | Magian + Oboro (RUN-specific) | no |
| Epeolatry | Aeonic (RUN-specific, group) | no |

**Key overlap:** the **SoA → Vagary** long pole and the **Dynamis-Tavnazia → Dyna-D** unlock you're
already doing for BLU **also unlock RUN's reforged tank sets**. You are not gating RUN separately — the
expensive unlock work is shared. Only the **Sparks floor, Ogma's Cape, and the Aettir/Epeolatry
weapons** are RUN-specific.

## Recommended build order

1. **Level 81 → 99** (trust-carry, /WAR, XP stack).
2. **Confirm Children of the Rune; do Dimidiation at 99.**
3. **Equip the Sparks/RoE DT floor** + start building **Ogma's Cape** from daily Ambuscade. Grab an
   **Ambuscade GS** as your weapon floor.
4. **Start joining easy content as a functional tank:** Ambuscade, then **Odyssey Sheol A/B/C** (you're
   running these anyway for BLU/gil — bring RUN sometimes to learn the kit and start Nyame).
5. **Grind the reforged sets over time:** **Futhark** via Dyna-D (no Vagary needed — can start as soon
   as Dyna-D is open) and **Erilaz** via Sortie (after the Vagary/SoA unlock that BLU also needs).
6. **Long-term:** **Nyame** (Odyssey) and eventually **Aettir → Epeolatry** as the capstone weapon.

**Bottom line:** RUN reaches "contributes to endgame content" quickly — **99 + Dimidiation + Sparks DT
floor + Ogma's Cape + an Ambuscade/Aettir GS** is enough to tank Ambuscade and Odyssey A/B. The
reforged JSE and Nyame then turn it from "functional" into "strong," largely off the back of unlock
work you're already doing for BLU.
